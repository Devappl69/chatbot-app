# chatbot-app
A Python application made with CustomTkinter that allows for chatting with chatbots

# Requirements
### Main
- Python 3.11
- Groq API Token *(get a free key here: https://console.groq.com/keys)*
### Libaries
- CustomTkinter 5.2.2
- Pillow 11.3.0
- Groq 0.29.0
